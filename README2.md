# Compilación

Para compilar necesita tener instalado ```gcc``` v5.1 o superior y ```make```
En windows ejecutar:

```
#bash
make -f makefile-W
```

En linux ejecutar:
```
#bash
make -f makefile-L
```

**Nota: No tiene soporte en mac**

# Ejecución

En consola ejecutar:
```
#bash
./app1 <data.csv> <metric1> <metric2> ... <metric10>
```
```
#bash
./app1 ventas.csv dms pls # other metrics
```
